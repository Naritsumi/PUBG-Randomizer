# Mi Excel - Loadout Randomizer

Una experiencia tactica para crear loadouts aleatorios de PUBG con estetica de operacion militar, mapas reales, armas oficiales y animaciones inspiradas en cajas de Counter-Strike.

El flujo esta pensado como si entraras a una partida: primero configuras el lobby, eliges si juegas solo o en squad, asignas tus operativos con avatares personalizados, seleccionas el mapa y finalmente despliegas los loadouts.

Cada jugador recibe una combinacion aleatoria de arma principal y secundaria segun el mapa elegido, con animaciones visuales distintas para Solo y Squad.

## Caracteristicas

- Seleccion de modo `Solo` o `Squad`.
- Configuracion de squad de 2, 3 o 4 jugadores.
- Avatares personalizados para cada operativo.
- Mapas reales de PUBG con imagenes oficiales.
- Armas reales con renders oficiales.
- Pool de armas adaptado al mapa seleccionado.
- Animacion tipo caja para modo Solo.
- Animacion tipo tragaperras/reel tactico para Squad.
- Interfaz progresiva: lobby, mapa y loot.
- Estetica verde/naranja inspirada en HUD militar.

## Experiencia

1. Configura el lobby.
2. Elige modo Solo o Squad.
3. Asigna nombres a los operativos.
4. Selecciona el mapa.
5. Prepara los loadouts.
6. Randomiza armas para todos.
